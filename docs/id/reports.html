Laporan¶

Laporkan tentang proyek skenario dampak yang dibuat menggunakan QGIS template komposer. Pengaturan bawaan template sudah menjadi bagian dari plugin, tapi anda dapat dengan mudah membuat template anda sendiri di QGIS dan menggunakannya sebagai dasar dalam laporan anda.

Template lokal didukung melalui skema sederhana berikut ini:

Jika ada sebuah template tertentu yang disebut /path/to/template/foo.qpt lalu plugin terlihat di direktori template /path/to/template/ untuk sebuah file memanggil file foo-LANG.qpt, dimana “LANG” adalah kode bahasa untuk sistem lokal.

Jika sebuah file ada maka akan digunakan untuk membuat laporan sebaliknya template asli /path/to/template/foo.qpt akan digunakan

Laporan Elemen Template¶

Template ini mencakup 3 kelompok elemen selanjutnya :

  • Elemen statis.

  • Elemen yang mencakup token-token untuk penggantian.

  • Elemen yang mana ter-update secara langsung oleh para penyaji.

Elemen Statis¶

Ini adalah elemen-elemen yang tidak diubah selama pembuatan laporan. seperti beberapa logo/gambar. teks tambahan, dll. Anda dapat menghapus beberapa elemen atau menggantinya dengan yang anda inginkan jika itu diperlukan.

Elemen yang mencakup token-token untuk penggantian¶

Dalam kasus ini elemen nama tidak signifikan, hanya sebuah (beberapa) tanda yang ada. Saat ini tanda-tanda yang ada di elemen akan diganti dengan konten yang telah diterjemahkan (jika alternatif lokal berlaku) dari plugin sesuai dengan kata kunci yang tertulis pada dokumen di bawah ini.

Elemen yang mana ter-update secara langsung oleh para penyaji¶

Dalam kasus ini semua konten yang mungkin hadir di elemen seluruhnya telah diganti oleh peta realtime, walaupun pilihan gaya tampilan ini (contoh pengaturan graticule pada peta) akan menyisakan efek tertentu

Elemen ini dikenali dengan ID-nya dan plugin saat ini mendukung elemen-elemen berikut:

  • safe-logo — QGIS composer image, which will be used for displaying InaSAFE logo with web-site url.
  • organisation-logo — QGIS composer image, which will be used for displaying organisation logo. By default a combined supporters logo is used. To use custom logo image, set the path to your PNG file in the plugin Options dialog (see Pilihan for details).

  • peta-dampak — Komposer peta QGIS, yang akan digunakan untuk menampilkan skenario dampak.

  • keterangan-dampak — Komposer Keterangan QGIS, yang akan digunakan untuk menampilkan keterangan dari skenario dampak.

  • impact-report — QGIS composer label, which will be used for displaying the impact report table. NOTE: as QGIS composer doesn’t support automatic resizing elementse, based on their contents, make sure that label have dimensions large enough to show full table.

Kata kunci yang dapat diganti¶

Sesi ini menjelaskan kata kunci tokenised yang dilewatkan di template peta. Untuk menyisipkan kata kunci apapun ke dalam template peta, sertakan kunci di [] (contoh [nama-tempat]) dan dia akan terganti dengan nilai teks (contoh Tondano). Daftar tersebut termasuk frasa statis yang telah diinternasionalisasi (dan karenanya akan ditampilkan dalam bahasa lokal peta tersebut, atur ke dalam bahasa inggris jika tidak ada terjemahan yang tersedia).

Kata kunci tokenized berikut ini didukung:

  • judul-dampak — judul dari peta skenario dampak.

  • tanggal — tanggal pengukuran.

  • waktu — waktu pengukuran.

  • versi-safe — versi plugin yang digunakan.

  • disclaimer — disclaimer text. By default used next text: “InaSAFE has been jointly developed by BNPB, Australian Government and the World Bank - GFDRR”. To use custom disclaimer text, enter desired text the plugin Options dialog (see Pilihan for details).

Template yang lazim digunakan¶

Anda memiliki beberapa pilihan untuk mengatur template tanpa melakukan proses pemprograman apapun. Ada 3 cara utama untuk melakukan ini:

  • Pindahkan kata kunci yang dapat diganti ke dalam elemen yang berbeda atau pindahkan seluruhnya.

  • Ubah tampilan elemen template (contoh mengatur huruf, warna, garis, dll). pindahkan elemen template itu sendiri di sekitarnya atau tambahkan/hapus seluruhnya.

  • Buat template anda sendiri dan gunakan template itu daripada template bawaan.

The default template is provided as inasafe/safe_qgis/resources/qgis-composer-templates/inasafe.qpt and can be modified by opening the template using the QGIS map composer, making your changes and then overwriting the template or saving it in any directory.

You should take care to test your template changes before using it in a production environment.

Also in addition to default template you can have as many custom templates as you want. Just create templates, put them in some folder somewhere in filesystem and go to Plugins ‣ InaSAFE ‣ InaSAFE Options or click on the InaSAFE plugin toolbar’s options icon. In Options dialog specify Report template setting. It should contain full path to the folder with your custom templates.
