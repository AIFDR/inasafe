Pilihan¶

InaSAFE plugin menyediakan dialog pilihan yang memungkinkan Anda untuk menentukan berbagai pilihan yang berhubungan untuk bagaimana InaSAFE akan berperilaku. Dialog pilihan dapat ditampilkan dengan mengklik pada InaSAFE simbol plugin toolbar pilihan (seperti yang terlihat di bawah) atau dengan melakukan Plugins ‣ InaSAFE ‣ InaSAFE Options.

Then this dialog will appear:

Dialog Pilihan

Berikut ini pilihan yang tersedia di dalam Options Dialog:

As you can see this dialog is divided into three tabs namely:

  • Basic Options
  • Template Options
  • Advanced

Basic Options¶

  • Only show visible layers in the InaSAFE dock : This option will determine whether all hazard and impact layers should be listed in the InaSAFE dock’s combo boxes or (when checked) only visible layers.
  • Set QGIS layer name from ‘title’ in keywords : This option will cause QGIS to name layers in the Layers tree using the title keyword in the layer’s keywords file. If the layer has no ‘title’ in its keywords, or it has no keywords at all, the normal QGIS behaviour for naming layers will apply.

  • Zoom to impact layer on scenario estimate completion : Pilihan ini akan menyebabkan tampilan peta untuk diperbesar/diperkecil sehingga benar-benar berisi InaSAFE hasil skenario peta dampak ketika analisis selesai.

  • Hide exposure layer on scenario estimate completion : Pilihan ini akan menyebabkan QGIS untuk mematikan layer keterpaparan yang sedang digunakan ketika InaSAFE melengkapi analisis ini. Anda dapat mengaktifkan kembali visibilitas layer lagi dengan mencentang kotak centang di legenda.

  • Clip datasets to visible extent before analysis : Pilihan ini akan menyebabkan QGIS memotong layer ancaman dan keterpaparan yang terlihat pada peta kanvas.

  • When clipping, also clip features (e.g. will clip polygon smaller): Pilihan ini, setiap poligon yang melampaui luasan analisis akan dipotong sehingga mereka berisi oleh luasan analisis. contohnya jika Anda memiliki layer poligon ancaman banjir yang meluas jauh melampaui area yang Anda minati, poligon banjir akan berkurang sehingga hanya bagian di dalam area yang Anda minati dipertahankan. Ini dapat mempercepat waktu pemrosesan.

  • Show intermediate layers generated by postprocessing : Pilihan ini akan mnyebabkan QGIS menampilkan file menengah yang dihasilkan oleh langkah-langkah postprocessing pada peta kanvas.

  • Female ratio default value : This is the default female to total population ratio used por generating Gender breakdowns

Template Options¶

  • Organisation logo: Pilihan ini digunakan untuk menentukan logo yang akan digunakan sebagai logo organisasi pada berkas pdf proyek skenario dampak anda. Lihat Laporan pada Panduan Pengguna InaSAFE untik informasi lebih lanjut mengenai laporan.

  • Report templates directory: This option is used to determine a directory with custom templates. Templates from this directory will be available for usage when generating report on your impact scenario project. See Laporan in the InaSAFE User-Manual for more information on the reports.
  • Organisation disclaimer text: This option is used to change default disclaimer text, printed in reports. Just enter your own text and it will be used instead of the default one. See Laporan in the InaSAFE User-Manual for more information on the reports.

Advanced¶

  • Keyword cache for remote datasources : Opsi ini digunakan untuk menentukan di mana kata kunci akan disimpan untuk dataset apabila tidak dimungkinakn untuk ditulis ke dalam file .keywords. Lihat Sistem Kata kunci pada Panduan Pengguna InaSAFE untuk informasi mengenai sistem kata kunci.

  • Help to improve InaSAFE by submitting errors to a remote server : Pilihan ini, ketika diaktifkan, akan mengirimkan informasi diagnosis kembali ke server proyek InaSAFE di setiap kesalahan kita mampu menjebaknya. Informasi yang diberikan berguna bagi tim InaSAFE untuk meningkatkan ketahanan perangkat lunak yang kami produksi dan untuk melihat tren pada isu-isu yang dihadapi orang. Silakan catat teks peringatan dibawah pilihan ini yang disertakan disini secara penuh:

Catatan

The above setting requires a QGIS restart to disable/enable. Error messages and diagnostic information will be posted to http://sentry.linfiniti.com/inasafe-desktop/ . Some institutions may not allow you to enable this feature - check with your network administrator if unsure. Although the data is submitted anonymously, the information contained in tracebacks may contain file system paths which reveal your identity or other information from your system.

  • Enable developer mode : With this option enabled, you can right click on the report area and use the webkit debug tools to debug the output. This option is intended for advanced users only.
  • Use QGIS zonal statistics : With this button you can toggle between using QGIS internal zonal statistics or the one that comes with InaSAFE

Catatan

Anda dapat klik tombol Help beberapa waktu dan ini akan membuka bantuan pencarian dokumentasi untuk halaman ini

Catatan

Dengan menekan Batal maka dialog pilihan akan menutup dan perubahan apa saja tidak akan diterapkan. Tekan OK maka kotak dialog Pilihan akan menutup dan perubahan apa saja akan segera diterapkan.

Catatan

Urutan tombol yang tepat yang ditampilkan pada kotak dialog ini mungkin berbeda tergantung pada sistem operasi atau lingkungan desktop Anda.
